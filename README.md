🏋️ Gym Management & Workout Tracker

A full-stack web application designed to manage gym operations and track user workouts efficiently.
Built with a modern tech stack including React, Node.js / Python backend, and MySQL.

🚀 Features -

  -🔐 User Authentication (Admin, Trainer, Member, Owner roles)
  -🏋️ Workout Tracking & History
  -📊 Dashboard for activity insights
  -👥 Gym Member Management
  -🧾 Membership Plans & Management
  -🧑‍🏫 Trainer Assignment
  -📅 Workout Scheduling
  -⚡ Real-time frontend with React


Steps to Run 

Create a project 
npm create vite@latest

- cd Repbase frontend
    
   - npm install 
   - npm install axios
     
In one Terminal
 - uvicorn api:app --reload

In second Terminal
 - cd repbase frontend
 - npm run dev
